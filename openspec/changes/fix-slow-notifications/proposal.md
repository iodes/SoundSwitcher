## Why

프로파일 변경시 발생하는 알림이 화면에 나타나고 사라지는 속도가 너무 느려서, 사용자가 프로파일을 연속적으로 변경할 경우 알림이 화면에 계속 쌓이는 문제가 있습니다. 이를 수정하여 기기 전환 시 더 즉각적이고 쾌적한 피드백을 제공하고자 합니다.

## What Changes

- 알림(Toast/Notification)의 표시 및 사라짐 애니메이션 시간(Duration)을 단축합니다.
- 새로운 알림이 발생할 때 이전에 떠 있던 알림을 즉시 닫거나 덮어쓰도록 하여 알림이 누적되어 쌓이는 것을 방지합니다.

### New Capabilities

### Modified Capabilities
- `device-switching`: 프로파일 전환 완료 후 제공되는 시각적 피드백(알림)의 응답성과 중복 방지 처리 요구사항이 변경됩니다.

## Impact

- 알림을 담당하는 UI 컴포넌트 또는 서비스 (예: `App.xaml`, 알림 윈도우/컨트롤 등)의 애니메이션, 타이머 동작 코드
- 기기 전환 시의 사용자 경험 (더 빠르고 직관적인 전환 피드백)
