## Why

메인 메뉴의 프로필 카드는 현재 마우스 오버(Hover)이나 클릭(Mouse Down) 시 시각적인 피드백이 부족하여 클릭 가능한 요소라는 점이 직관적으로 다가오지 않습니다. 또한, 프로필 아이콘을 직접 클릭하여 아이콘을 변경하는 기능이 카드 내 공간을 차지하고 있어, 카드를 클릭하여 프로필을 전환할 수 있는 유효 클릭 영역이 좁아지는 문제가 있습니다. 이를 개선하여 사용성을 높이고자 합니다.

## What Changes

- 메인 메뉴의 프로필 카드에 마우스 오버 및 마우스 다운 시 시각적 클릭 효과(배경색 변화, 미세한 크기 변화 등)를 추가합니다.
- 기존에 프로필 아이콘을 직접 클릭하여 아이콘을 변경하던 기능을 제거합니다.
- 프로필 카드의 우클릭 컨텍스트 메뉴에 "아이콘 변경" 항목을 새롭게 추가합니다.
- 아이콘 클릭 이벤트가 제거됨에 따라, 프로필 카드 전체 영역을 온전히 프로필 전환을 위한 클릭 영역으로 활용하도록 클릭 영역을 확장합니다.

### Modified Capabilities

## Impact

- 메인 메뉴 UI의 프로필 카드 관련 컴포넌트 디자인 및 상호작용 로직
- 프로필 아이콘 변경 트리거 로직 및 우클릭 컨텍스트 메뉴 구성 로직
