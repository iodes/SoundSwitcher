## MODIFIED Requirements

### Requirement: 한 번의 클릭으로 장치 순환 전환
시스템 트레이 아이콘을 좌클릭하면 즉시 유저가 설정한 순서대로 다음 **프로파일**로 전환되어야 한다. 단, 등록된 프로파일이 0개일 경우 메인 설정 창이 열려야 하며, 등록된 프로파일이 1개일 경우 전환 동작을 무시해야 한다.

#### Scenario: 좌클릭 프로파일 전환
- **WHEN** 사용자가 트레이 아이콘을 마우스 왼쪽 버튼으로 클릭할 때 (프로파일 2개 이상)
- **THEN** 설정된 순서에 따라 다음 프로파일로 활성화 상태가 넘어가고, 연결된 입/출력 장치가 동시 변경되며, 트레이 아이콘 이미지와 툴팁이 즉시 갱신되어야 한다

#### Scenario: 등록된 프로파일이 없을 때 좌클릭
- **WHEN** 사용자가 트레이 아이콘을 마우스 왼쪽 버튼으로 클릭할 때, 등록된 프로파일이 0개인 경우
- **THEN** 다음 프로파일로 전환을 시도하지 않고 메인 설정 창이 열려야 한다

#### Scenario: 등록된 프로파일이 1개일 때 좌클릭
- **WHEN** 사용자가 트레이 아이콘을 마우스 왼쪽 버튼으로 클릭할 때, 등록된 프로파일이 1개인 경우
- **THEN** 불필요한 장치 전환을 시도하지 않고 아무 동작도 수행하지 않아야 한다
