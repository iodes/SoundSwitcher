## Why

트레이 아이콘이 Windows 시스템 트레이의 숨겨진 아이콘 영역(접힘 메뉴) 안에 위치할 때, 아이콘을 우클릭하면 컨텍스트 메뉴가 마우스 커서 위치나 아이콘 위치가 아닌 잘못된 위치에 표시되는 버그가 있습니다. 사용자가 메뉴 항목을 정상적으로 클릭할 수 없게 만들어 사용성에 심각한 지장을 주므로 신속한 수정이 필요합니다.

## What Changes

- 트레이 아이콘 우클릭 시 표시되는 컨텍스트 메뉴의 좌표 계산 로직을 수정합니다.
- 아이콘이 숨겨진 트레이 영역에 있을 때도 마우스 커서의 현재 위치를 정확히 가져와 해당 위치에 메뉴가 렌더링되도록 개선합니다.

## Capabilities

### New Capabilities
(없음)

### Modified Capabilities
- `tray-interface`: 트레이 아이콘 컨텍스트 메뉴 표시 위치 정확도 개선

## Impact

- `SoundSwitcher` 애플리케이션의 트레이 메뉴 UI 렌더링 로직
- 의존성이나 다른 시스템에는 영향을 주지 않는 단순 UI 버그 수정입니다.
