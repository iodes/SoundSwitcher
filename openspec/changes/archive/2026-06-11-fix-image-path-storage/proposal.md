## Why

현재 사용자가 프로파일의 커스텀 이미지를 선택할 때, 상대적인 캐시 경로가 아닌 절대 경로가 설정에 저장됩니다. 이로 인해 앱의 위치가 변경되거나 다른 기기 간에 프로파일 설정이 동기화될 경우 프로파일 이미지가 깨지는 문제가 발생합니다.

## What Changes

- 이미지 저장 로직을 수정하여 설정에 절대 경로 대신 캐시 파일명(또는 상대 경로)만 저장하도록 변경합니다.
- 이미지 로드 로직을 수정하여 저장된 파일명을 기반으로 실제 절대 경로를 동적으로 해석하여 이미지를 표시하도록 보완합니다.

### Modified Capabilities
- `profile-management`: 아이콘의 경로를 저장할 때 절대 경로가 아닌 파일 이름/상대 경로를 저장하도록 요구사항이 변경되었습니다.

## Impact

- 프로파일 이미지 저장 및 불러오기 로직
- 프로파일 설정 저장 포맷 (이제 이미지 절대 경로 대신 파일명이 저장됨)
