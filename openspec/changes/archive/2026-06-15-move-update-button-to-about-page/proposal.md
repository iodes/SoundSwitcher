## Why

현재 트레이 아이콘의 컨텍스트 메뉴에 업데이트 확인 버튼이 위치해 있으나, 앱 버전 정보 등이 표시되는 About 페이지에서 업데이트를 확인하는 것이 사용자 경험상 더 직관적이고 자연스럽습니다. 트레이 메뉴의 복잡도를 줄이고 설정 페이지의 역할을 명확히 하기 위해 이 변경이 필요합니다.

## What Changes

- 트레이 아이콘 컨텍스트 메뉴에서 '업데이트 확인' 항목 제거
- 설정 창의 AboutPage(정보 페이지)에 '업데이트 확인' 버튼 추가 및 로직 연동

## Capabilities

### New Capabilities
- `about-page`: 앱의 정보(버전 등)를 표시하고 수동으로 업데이트를 확인할 수 있는 정보 페이지 제공 기능

### Modified Capabilities
- `tray-interface`: 트레이 아이콘 컨텍스트 메뉴 항목에서 업데이트 확인 기능이 제거됨

## Impact

- 트레이 메뉴 UI (컨텍스트 메뉴 항목 변경)
- 설정 창 UI (AboutPage 추가/수정)
- 업데이트 확인 로직을 호출하는 진입점 변경 (Tray -> AboutPage)
