## ADDED Requirements

### Requirement: 앱 정보 표시 및 수동 업데이트 확인
AboutPage(정보 페이지)에서는 현재 애플리케이션의 버전 정보를 표시하고, 사용자가 직접 최신 업데이트가 있는지 확인할 수 있는 '업데이트 확인' 버튼을 제공해야 한다.

#### Scenario: 업데이트 확인 버튼 클릭
- **WHEN** 사용자가 설정 창의 AboutPage에서 '업데이트 확인' 버튼을 클릭할 때
- **THEN** 어플리케이션은 서버와 통신하여 최신 업데이트가 있는지 확인하는 로직을 수행하고, 결과에 따라 사용자에게 알림 또는 설치 UI를 제공해야 한다.
