## ADDED Requirements

### Requirement: 오디오 장치 목록 관리 및 입/출력 구분
시스템은 시스템에 연결된 전체 오디오 장치 목록을 '출력 장치'와 '입력 장치'로 구분하여 제공하고, 사용자가 전환할 '선호 장치'들을 체크박스로 선택할 수 있어야 한다.

#### Scenario: 체크박스를 통한 선호 장치 등록
- **WHEN** 사용자가 설정 창의 '장치 탭'에서 출력 및 입력 장치 목록 중 원하는 장치의 체크박스를 선택하고 저장할 때
- **THEN** 시스템은 체크된 장치들로만 구성된 활성화 목록을 사용자 설정에 저장해야 한다

### Requirement: 장치 순서 커스터마이징
시스템은 사용자가 체크한 선호 장치들의 순서를 임의로 재배치할 수 있도록 허용해야 한다.

#### Scenario: 드래그 앤 드롭으로 순서 변경
- **WHEN** 사용자가 설정 창에서 선호 장치 항목을 드래그하여 순서를 변경할 때
- **THEN** 변경된 순서가 사용자 설정에 저장되고, 트레이 클릭 순환 로직에 반영되어야 한다

### Requirement: 개별 장치 아이콘 설정
시스템은 사용자가 각 장치에 대해 커스텀 아이콘을 지정할 수 있도록 해야 한다.

#### Scenario: 장치 아이콘 변경
- **WHEN** 사용자가 특정 장치의 아이콘 설정 버튼을 눌러 새로운 아이콘 이미지를 지정할 때
- **THEN** 해당 아이콘 경로가 설정에 저장되고, 장치가 활성화될 때 트레이 아이콘에 반영되어야 한다

### Requirement: 선호 장치 순환 전환 (사용자 지정 순서)
전환 명령이 발생하면, 시스템은 사용자가 배치한 순서에 따라 다음 선호 장치로 기본 오디오 장치를 변경해야 한다.

#### Scenario: 지정된 순서대로 전환
- **WHEN** 사용자가 '헤드셋 -> 스피커 -> 이어폰' 순서로 배치한 상태에서 전환 명령이 발생할 때
- **THEN** 현재 장치가 '헤드셋'이면 '스피커'로, '스피커'이면 '이어폰'으로 순차적으로 변경되어야 한다

### Requirement: 기본 통신 장치 동시 전환 설정
시스템은 장치 전환 시 '기본 장치'만 변경할지, '기본 통신 장치'도 함께 변경할지 설정 탭에서 옵션을 제공해야 한다.

#### Scenario: 통신 장치 동시 전환
- **WHEN** 사용자가 설정 탭에서 '통신 장치 동시 전환' 옵션을 켜고 트레이 아이콘을 클릭하여 전환할 때
- **THEN** 다음 장치가 시스템의 '기본 재생 장치'와 '기본 통신 장치'로 모두 설정되어야 한다
