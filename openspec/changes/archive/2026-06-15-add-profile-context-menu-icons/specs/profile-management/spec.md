## MODIFIED Requirements

### Requirement: 컨텍스트 메뉴를 통한 프로파일 관리 기능
프로파일 카드 내의 오동작 방지 및 기능 응집을 위해, 프로파일 이름 변경, 복제, 삭제, 아이콘 변경, 기본 아이콘으로 초기화, 기본값으로 설정 등 관리 기능은 카드 전체 영역에 대한 우클릭 컨텍스트 메뉴를 통해 제공되어야 한다. 사용자가 메뉴 항목을 빠르고 직관적으로 식별할 수 있도록 컨텍스트 메뉴의 각 항목에는 알맞은 기능의 아이콘이 함께 표시되어야 한다.

#### Scenario: 삭제 및 관리 컨텍스트 메뉴 호출 및 아이콘 표시
- **WHEN** 사용자가 프로파일 카드 위에서 우클릭할 때
- **THEN** 프로파일 카드 위치 근처에 관리 옵션이 포함된 컨텍스트 메뉴가 나타나야 한다
- **AND** 컨텍스트 메뉴의 각 항목(삭제, 아이콘 변경 등) 왼쪽에는 항목의 의미를 시각적으로 나타내는 아이콘이 표시되어야 한다

#### Scenario: 메뉴를 통한 삭제 실행 및 캐시 삭제
- **WHEN** 사용자가 컨텍스트 메뉴의 '삭제' 옵션을 좌클릭할 때
- **THEN** 해당 프로파일이 목록에서 즉시 제거되며, 해당 프로파일이 커스텀 아이콘을 사용 중이었다면 앱 데이터 폴더 내의 연관된 캐시 이미지 파일도 물리적으로 삭제되어야 한다
