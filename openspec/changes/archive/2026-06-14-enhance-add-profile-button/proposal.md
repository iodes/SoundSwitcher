## Why

사용자가 프로파일을 추가할 때 "프로파일 추가 버튼"에 대한 시각적인 피드백(마우스 오버, 클릭 효과)이 부족하여 상호작용이 다소 밋밋하게 느껴집니다. 테두리를 좀 더 부드럽게(흐리게) 만들고 UI에 자연스럽게 녹아들도록 개선하여 더 나은 사용자 경험을 제공하고자 합니다.

## What Changes

- 프로파일 추가 버튼 마우스 오버 시 시각적 효과 (예: 배경색 변화, 투명도 조절 등) 추가
- 프로파일 추가 버튼 클릭 시 시각적 효과 (예: 크기 축소 애니메이션 등) 추가
- 버튼 테두리 스타일 변경 (테두리를 흐리게 하고 둥글게 처리하여 UI에 자연스럽게 녹아들도록 변경)

### New Capabilities

### Modified Capabilities

- 메인 화면 내 프로파일 추가 버튼 (아마도 `MainPage.xaml` 또는 해당 버튼이 포함된 뷰)
- 해당 버튼의 스타일 (XAML)
