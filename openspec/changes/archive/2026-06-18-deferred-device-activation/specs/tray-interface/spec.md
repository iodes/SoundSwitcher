## MODIFIED Requirements

### Requirement: 시스템 트레이 커스텀 아이콘 표시
애플리케이션이 실행 중일 때 시스템 트레이에 상주하며 현재 활성화된 **프로파일**의 아이콘을 표시해야 한다. 유저가 커스텀 아이콘을 설정한 경우 해당 아이콘이, 그렇지 않은 경우 기본 아이콘이 표시되어야 한다. 사용자가 장치 활성화를 대기 중인 상태일 때는 기존의 트레이 아이콘 이미지를 그대로 유지하되, 트레이 아이콘에 마우스 오버 시 표시되는 툴팁(Tooltip) 텍스트를 변경하여 현재 전환 대기 중인 프로파일에 대한 정보를 표기해야 한다.

#### Scenario: 활성 프로파일 툴팁 및 아이콘 렌더링
- **WHEN** 현재 시스템 장치가 특정 프로파일의 설정과 일치할 때
- **THEN** 트레이 아이콘은 해당 프로파일의 아이콘으로 표시되며, 마우스 오버 시 프로파일 이름이 툴팁으로 표시되어야 한다

#### Scenario: 기본 장치 아이콘 렌더링 (Fallback)
- **WHEN** 현재 활성 장치가 커스텀 아이콘이 설정되지 않은 장치로 전환될 때
- **THEN** 트레이 아이콘 이미지가 시스템에 내장된 Windows 기본 장치 아이콘으로 표시되어야 한다

#### Scenario: 대기 상태의 프로파일 트레이 툴팁 표시
- **WHEN** 사용자가 전환을 시도한 프로파일의 장치가 비활성 상태여서 '대기(Pending)' 상태로 전환되었을 때
- **THEN** 시스템 트레이의 아이콘 이미지는 변경하지 않고 유지하되, 마우스 오버 시 표시되는 툴팁 텍스트에 "대기 중: [프로파일 이름]" 등 전환 대기 상태임을 안내하는 내용을 표시해야 한다
