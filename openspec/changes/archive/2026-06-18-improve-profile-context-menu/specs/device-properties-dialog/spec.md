## ADDED Requirements

### Requirement: 오디오 장치 네이티브 속성 창 호출
애플리케이션은 특정 오디오 장치(출력 또는 입력)에 대해 Windows에서 기본으로 제공하는 네이티브 장치 속성 창을 호출할 수 있어야 한다.

#### Scenario: 출력 장치 속성 창 성공적 호출
- **WHEN** 사용자가 프로파일 컨텍스트 메뉴에서 유효한 출력 장치가 설정된 상태로 '출력 장치 속성' 항목을 클릭할 때
- **THEN** 해당 출력 장치의 Windows 네이티브 속성 창이 화면에 나타나야 한다

#### Scenario: 입력 장치 속성 창 성공적 호출
- **WHEN** 사용자가 프로파일 컨텍스트 메뉴에서 유효한 입력 장치가 설정된 상태로 '입력 장치 속성' 항목을 클릭할 때
- **THEN** 해당 입력 장치의 Windows 네이티브 속성 창이 화면에 나타나야 한다

#### Scenario: 유효하지 않은 장치의 속성 호출 시도
- **WHEN** 오디오 장치가 존재하지 않거나, 시스템에서 삭제된 장치에 대해 속성 창 호출을 시도할 때
- **THEN** 속성 창이 열리지 않아야 하며, 앱이 충돌 없이 안전하게 이벤트를 무시하거나 적절한 오류 메시지(또는 폴백으로 기본 소리 제어판)를 처리해야 한다
