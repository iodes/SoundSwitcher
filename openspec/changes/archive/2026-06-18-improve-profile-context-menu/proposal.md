## Why

현재 프로파일의 컨텍스트 메뉴는 구조가 다소 혼란스럽고, 장치 설정을 쉽게 확인하거나 변경하기 위한 접근성이 부족합니다. 사용자가 프로파일에 연결된 오디오 장치의 설정을 쉽게 접근할 수 있도록 윈도우 네이티브 장치 속성 창을 호출하는 기능을 추가하고, 메뉴 항목들을 논리적인 그룹으로 분리하여 직관적인 사용성을 제공하기 위함입니다.

## What Changes

- 프로파일 컨텍스트 메뉴 항목을 다음과 같이 재구성하고 디바이더로 그룹화합니다:
  - 기본 프로파일 설정
  - (디바이더)
  - 아이콘 관련 메뉴들
  - (디바이더)
  - 출력 장치 속성 (Windows 네이티브 장치 속성 창 호출)
  - 입력 장치 속성 (Windows 네이티브 장치 속성 창 호출)
  - (디바이더)
  - 삭제
- **새로운 기능**: 컨텍스트 메뉴에서 선택된 프로파일의 출력/입력 장치에 대한 Windows 네이티브 장치 속성 창을 여는 기능 추가.

## Capabilities

### New Capabilities
- `device-properties-dialog`: Windows 네이티브 오디오 장치 속성 창을 호출하는 기능

### Modified Capabilities
- `profile-management`: 프로파일 컨텍스트 메뉴의 구조 및 항목 변경
- `tray-interface`: UI 상에서 프로파일 컨텍스트 메뉴의 레이아웃 재구성 및 커맨드 바인딩

## Impact

- 트레이 영역의 프로파일 컨텍스트 메뉴 UI 구성 코드 (`MainWindow` 또는 관련 뷰)
- 장치 속성 창을 띄우기 위한 Windows API 상호 작용 코드 추가
- 프로파일 관리에 필요한 ViewModel 커맨드 추가 (출력/입력 장치 속성 열기)
