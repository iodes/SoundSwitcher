## Why

사용자가 자주 사용하는 특정 프로파일을 기본 프로파일로 지정하여, 프로그램(윈도우) 시작 시 항상 해당 프로파일로 자동 전환되도록 하여 편의성을 높이기 위함입니다. 이전 종료 시 다른 프로파일을 사용했더라도 시작 시 기본 프로파일로 돌아올 수 있게 합니다.

## What Changes

- 프로파일 우클릭 컨텍스트 메뉴에 "기본 프로파일로 설정" 및 "기본 프로파일 설정 해제" 옵션 추가 (다국어 지원 포함, "아이콘 변경" 메뉴 위쪽에 위치).
- 단일 기본 프로파일 유지: 다른 프로파일을 기본 프로파일로 설정하면 기존 기본 프로파일은 자동으로 해제됨.
- 기본 프로파일 설정 저장 및 관리 로직 추가.
- 프로그램 시작 시 설정된 기본 프로파일과 마지막으로 사용된(현재) 프로파일을 비교하여 다를 경우 기본 프로파일로 전환하는 로직 추가. (프로파일 동일성 판정에는 공용 로직을 사용)

## Capabilities

### New Capabilities
- `default-profile`: 기본 프로파일 설정/해제 및 시작 시 자동 전환 기능

### Modified Capabilities

## Impact

- 프로파일 컨텍스트 메뉴 UI 코드 (다국어 메뉴 항목 추가: 설정 및 해제)
- 프로그램 초기화/시작 로직 (기본 프로파일 확인 및 전환)
- 설정 저장 로직 (기본 프로파일 설정 데이터 저장)
