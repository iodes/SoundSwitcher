## MODIFIED Requirements

### Requirement: 프로파일 카드 클릭을 통한 활성화
사용자는 프로파일 카드의 빈 영역(아이콘 영역, 콤보박스, 드래그 핸들 제외)을 좌클릭하여 해당 프로파일을 활성화(전환)할 수 있어야 하며, 전환 시 사용자 설정에 따라 프로파일 변경 알림이 표시되어야 한다.

#### Scenario: 일반 영역 좌클릭을 통한 전환 및 알림 표시
- **WHEN** 사용자가 프로파일 카드의 정보가 표시되는 일반 영역(빈 배경 등)을 좌클릭할 때
- **THEN** 해당 카드의 프로파일이 즉시 활성화(선택)되고 사운드 재생 장치가 해당 프로파일에 지정된 장치로 전환되어야 한다
- **AND** '프로파일 변경 알림 표시' 설정이 켜져 있는 경우, 화면에 프로파일 변경 알림이 표시되어야 한다

#### Scenario: 내부 컨트롤과 상호작용 시 전환 방지
- **WHEN** 사용자가 프로파일 카드 내부의 아이콘 영역, 연결 장치 선택 콤보박스, 혹은 좌측 드래그 핸들을 좌클릭할 때
- **THEN** 각각의 컨트롤 고유 기능만 수행되며, 프로파일 전환 이벤트는 발생하지 않아야 한다

## ADDED Requirements

### Requirement: 프로파일 변경 알림 표시 설정
사용자는 애플리케이션 설정을 통해 프로파일 변경 시 알림 표시 여부를 제어할 수 있어야 한다. 이 설정은 핫키 및 메인 UI를 통한 모든 프로파일 전환 방법에 공통으로 적용된다.

#### Scenario: 알림 설정 비활성화 상태에서 전환
- **WHEN** 사용자가 '프로파일 변경 알림 표시' 설정을 끄고 핫키나 프로파일 카드를 클릭하여 프로파일을 전환할 때
- **THEN** 프로파일 및 오디오 장치 전환은 정상적으로 수행되지만, 변경 안내 알림은 화면에 표시되지 않아야 한다
