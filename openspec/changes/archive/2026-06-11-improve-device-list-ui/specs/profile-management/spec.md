## ADDED Requirements

### Requirement: 장치 콤보박스 목록 정렬
프로파일 카드 내부의 장치 선택 콤보박스는 드롭다운 목록을 표시할 때 항상 오디오 장치의 이름을 기준으로 알파벳순(ABC순, 오름차순)으로 정렬하여 표시해야 한다.

#### Scenario: 콤보박스 드롭다운 확인 시 정렬
- **WHEN** 사용자가 프로파일 카드 내의 장치 선택 콤보박스를 클릭하여 드롭다운 목록을 열었을 때
- **THEN** 목록 내의 장치들이 모두 알파벳순(ABC순)으로 오름차순 정렬되어 나타나야 한다

### Requirement: 장치 콤보박스 선택 항목 텍스트 말줄임 처리
장치 선택 콤보박스의 선택된 장치 이름이 콤보박스의 가로 너비보다 긴 경우, 텍스트의 끝 부분을 '...'으로 말줄임(Character Ellipsis) 처리하여 UI 레이아웃이 깨지지 않도록 해야 한다.

#### Scenario: 긴 이름의 장치가 선택되었을 때
- **WHEN** 이름이 매우 긴 오디오 장치가 콤보박스에 선택되어 표시될 때
- **THEN** 콤보박스 영역을 벗어나는 텍스트는 잘리고 끝에 '...'이 표시되어야 한다

### Requirement: 장치 콤보박스 목록 내 아이콘 표시
콤보박스의 드롭다운을 펼쳐 장치 목록을 확인할 때, 각 텍스트 항목 좌측에 해당 장치의 시각적 식별을 돕는 장치 아이콘을 함께 표시해야 한다.

#### Scenario: 콤보박스 드롭다운 펼침 시 아이콘 확인
- **WHEN** 사용자가 콤보박스를 클릭하여 드롭다운 목록을 펼쳤을 때
- **THEN** 목록 내의 각 장치 이름 텍스트 옆에 해당 장치 유형(예: 스피커, 마이크, 헤드셋 등)을 나타내는 아이콘이 표시되어야 한다
