## 2. 장치 탭 UI 전면 개편

- [x] 2.1 하단에 점선 테두리를 가진 '새 프로파일 추가 +' 버튼 구현 (클릭 시 카드가 쌓이고 버튼은 아래로 밀려나는 스택 구조)
- [x] 2.2 각 프로파일을 아름다운 카드 UI 형태로 구성: 출력 장치 및 입력 장치를 페어로 명확히 표시
- [x] 2.3 각 카드 좌측에 네이티브 WPF UI 핸들 그립을 추가하여 직관적인 드래그 앤 드롭 순서 변경 로직 및 뷰 연동 구현
- [x] 2.4 기존 체크박스 UI를 제거하고, 카드에 마우스 오버(Hover) 시 나타나는 휴지통 모양의 삭제 버튼 및 삭제 기능 구현
- [x] 2.5 수동 새로고침 버튼을 제거하고, `MMDeviceEnumerator`의 장치 상태 변경 이벤트를 구독하여 연결/해제 시 장치 콤보박스 목록이 자동 갱신되도록 구현

## 3. 트레이 전환 로직 및 피드백 수정

- [x] 3.1 트레이 좌클릭 시 순환되는 로직을 `DeviceProfile` 기반으로 수정 (지정된 출력 장치와 입력 장치를 동시에 변경 시도)
- [x] 3.2 일부 장치가 연결 해제되어 찾을 수 없는 경우, 가능한 장치만 변경하고 예외를 무시하는 안전한 전환 로직 구현
- [x] 3.3 트레이 아이콘 이미지를 현재 활성화된 `DeviceProfile`의 `IconPath`로 로드하도록 갱신 로직 수정
- [x] 3.4 트레이 아이콘 툴팁에 현재 활성화된 '프로파일 이름'이 표시되도록 연동
