## MODIFIED Requirements

### Requirement: 컨텍스트 메뉴 제공
시스템 트레이 아이콘을 우클릭하면 컨텍스트 메뉴가 표시되어야 한다. 이때 컨텍스트 메뉴는 마우스 포인터의 위치를 기준으로 정확하게 렌더링되어야 하며, 아이콘이 숨겨진 트레이 영역(접힘 상태)에 있을 때나 시스템 DPI 스케일이 100%가 아닐 때에도 화면 위치가 어긋나지 않아야 한다. 또한 메뉴에는 '시스템 소리 설정', '설정', '종료' 항목이 포함되어 있어야 하며, '업데이트 확인' 항목은 포함되지 않는다.
메뉴가 표시될 때에는 시스템의 기본 플라이아웃(슬라이드) 애니메이션을 사용하지 않고, 즉시 표시되거나 단순한 페이드 효과만 사용하여 반응성을 높여야 한다.

#### Scenario: 우클릭 설정 진입
- **WHEN** 사용자가 트레이 아이콘을 우클릭할 때
- **THEN** '시스템 소리 설정', '설정' 및 '종료' 옵션이 포함된 컨텍스트 메뉴가 열려야 하며, '업데이트 확인' 항목은 노출되지 않아야 한다.

#### Scenario: 숨겨진 트레이 아이콘 우클릭 시 컨텍스트 메뉴 위치 보정
- **WHEN** 사용자가 Windows의 '숨겨진 아이콘 표시' 메뉴 내에 위치한 트레이 아이콘을 우클릭할 때
- **THEN** 컨텍스트 메뉴가 잘못된 위치(예: 화면 좌상단 등)가 아닌 마우스 커서의 현재 위치에 정확히 표시되어야 하며, 다른 영역 클릭 시 메뉴가 정상적으로 닫혀야 한다

#### Scenario: 시스템 소리 설정 실행
- **WHEN** 컨텍스트 메뉴에서 '시스템 소리 설정'을 클릭할 때
- **THEN** Windows 기본 소리 제어판(mmsys.cpl) 창이 열려야 한다

#### Scenario: 컨텍스트 메뉴 애니메이션 제한
- **WHEN** 사용자가 트레이 아이콘을 우클릭하여 컨텍스트 메뉴가 열릴 때
- **THEN** 메뉴가 슬라이드/플라이아웃 애니메이션 없이 즉시 표시되거나 페이드 효과로만 나타나야 한다
