## ADDED Requirements

### Requirement: 앱 시작 시 시스템 언어 감지
앱이 처음 실행될 때, 사용자의 운영체제 기본 언어를 감지하여 지원하는 언어 중 하나로 설정해야 합니다. (기본 지원 언어: 영어, 일본어, 한국어, 중국어 간체, 중국어 번체) 매칭되는 언어가 없을 경우 영어(en-US)를 기본값으로 사용합니다.

#### Scenario: 시스템 언어가 한국어인 경우
- **WHEN** 앱을 처음 시작하고 운영체제 언어가 `ko`로 설정되어 있을 때
- **THEN** 앱 UI 언어가 자동으로 `ko-KR`로 설정되어 표시됩니다.

#### Scenario: 지원하지 않는 시스템 언어인 경우
- **WHEN** 앱을 처음 시작하고 운영체제 언어가 스페인어(`es`) 등으로 설정되어 있을 때
- **THEN** 지원하지 않는 언어이므로 기본 언어인 영어(`en-US`)로 앱 UI가 설정됩니다.

### Requirement: 설정 메뉴에서 언어 변경
사용자는 앱의 설정 메뉴에서 지원하는 언어 목록을 보고, 그 중 하나를 선택하여 앱의 UI 언어를 실시간으로 변경할 수 있어야 합니다.

#### Scenario: 한국어에서 영어로 언어 변경
- **WHEN** 설정 메뉴의 언어 콤보박스에서 'English'를 선택했을 때
- **THEN** 즉시 앱 내의 모든 UI 텍스트(메뉴, 버튼, 레이블 등)가 영어로 전환됩니다.

### Requirement: 선택한 언어 설정 저장 및 로드
사용자가 설정 메뉴에서 변경한 언어는 설정 파일에 저장되어야 하며, 다음 앱 실행 시 해당 언어가 적용되어야 합니다.

#### Scenario: 앱 재시작 시 언어 유지
- **WHEN** 사용자가 언어를 '日本語(일본어)'로 설정하고 앱을 재시작했을 때
- **THEN** 앱이 일본어 UI 상태로 로드됩니다.
