## Why

현재 트레이 메뉴에서 Windows 기본 시스템 소리 설정 화면으로 빠르게 접근할 수 있는 방법이 없습니다. 사용자가 시스템의 세부적인 시스템 소리 설정(재생/녹음 장치 관리 등)을 변경하고 싶을 때 더 나은 편의성을 제공하기 위해 클래식 시스템 소리 설정 창(mmsys.cpl)을 여는 바로가기를 추가해야 합니다.

## What Changes

- 시스템 트레이 아이콘 우클릭 시 나타나는 컨텍스트 메뉴에 "시스템 소리 설정" 항목 추가
- 해당 항목 클릭 시 Windows의 클래식 소리 제어판(`mmsys.cpl`) 실행 기능 구현
- (선택 사항) 메뉴 구분을 위해 "시스템 소리 설정" 항목과 다른 항목들 사이에 구분선(Separator) 추가

## Capabilities

### New Capabilities

### Modified Capabilities
- `tray-interface`: 트레이 컨텍스트 메뉴에 "시스템 소리 설정" 메뉴 항목 및 실행 동작 추가

## Impact

- 트레이 아이콘 컨텍스트 메뉴 UI 구조 변경
- 외부 프로세스(`mmsys.cpl`) 실행을 위한 코드 추가
