## Why

현재 프로그램은 단일 언어만 지원하여 다양한 국가의 사용자가 이용하기에 불편함이 있습니다. 릴리스 노트에서 여러 언어(영어, 일본어, 한국어, 중국어 간체, 중국어 번체)를 지원하고 있으므로, 프로그램 UI도 이에 맞춰 다국어를 지원하여 글로벌 사용자 경험을 향상시키고자 합니다.

## What Changes

- UI 다국어 지원을 위한 `LocalizationManager` 도입 (GestureWheel 저장소 방식 참조)
- 지원 언어 추가: 영어(en-US), 한국어(ko-KR), 일본어(ja-JP), 중국어 간체(zh-CN), 중국어 번체(zh-TW)
- 설정 UI에 언어 선택 옵션 추가
- 기존 하드코딩된 UI 문자열을 `.resx` 리소스 파일로 분리 및 데이터 바인딩 적용

## Capabilities

### New Capabilities
- `multilingual-support`: 프로그램 내 다국어 지원 및 설정 기능

### Modified Capabilities

## Impact

- 모든 하드코딩된 UI 문자열이 리소스 참조 방식으로 변경됩니다.
- 설정 항목에 사용자가 선택한 언어를 저장하기 위한 필드가 추가됩니다.
- 앱 시작 시 시스템 언어를 감지하여 기본 언어로 설정하는 로직이 추가됩니다.
