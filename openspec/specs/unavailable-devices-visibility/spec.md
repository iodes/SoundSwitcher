# Capability: unavailable-devices-visibility

## Purpose
현재 모든 장치가 콤보박스에 노출되어 발생하는 시각적 혼잡도를 줄이기 위해, 기본적으로 연결이 해제되었거나 사용할 수 없는 오디오 장치를 목록에서 숨기도록 동작을 변경합니다. 단, 사용자가 필요시 시스템 목록에서 해당 장치들을 다시 확인하고 프로필에 미리 할당할 수 있도록 토글 옵션을 제공합니다.

## ADDED Requirements

### Requirement: 사용할 수 없는 장치 표시 설정 항목 제공
설정 창의 "기본 통신 장치 전환" 하단에 "사용할 수 없는 장치 표시"를 켜고 끌 수 있는 UI 컨트롤(체크박스 또는 토글)이 제공되어야 한다. 이 설정의 기본값은 비활성화(꺼짐) 상태여야 한다.

#### Scenario: 설정 조작 및 즉각적 반영
- **WHEN** 사용자가 설정 메뉴에서 "사용할 수 없는 장치 표시" 옵션을 변경할 때
- **THEN** 해당 변경 사항이 애플리케이션 설정 파일에 즉시 저장되어야 한다
- **AND** 현재 UI의 모든 장치 콤보박스 목록 렌더링에 변경된 필터링 로직이 즉시 반영되어 갱신되어야 한다
