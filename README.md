# SoundSwitcher

SoundSwitcher is a Windows WPF application that allows you to easily manage and switch between different audio playback and capture device profiles.

## Features
- **Profile Management**: Create custom profiles for different audio setups (e.g., Gaming, Chat, Speakers).
- **Device Assignment**: Assign specific playback and capture devices to each profile.
- **Custom Icons**: Personalize profile icons for easy identification.
- **System Tray Integration**: Quickly switch active profiles from the system tray menu without opening the main window.
- **Reorder Profiles**: Drag and drop to reorder profiles based on your preference.

## Requirements
- Windows OS
- .NET runtime (WPF supported)
